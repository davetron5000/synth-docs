Features

Invert LFO Waveform

  1. Hold down Shift, turn Int.

LFO Syncing

  1. Enter Program Edit Mode, then tap 7

  2. Key sync resets the LFO on each note

  3. Voice sync keeps all four voices of polyphony on the same cycle

LFO Target

  1. Enter Program Edit Mode, then tap 7

  2. Choose the target of the LFO's modulation

    Value Target
    All All three oscillators
    VCO1+2 The two analog oscillators
    VCO2 Only VCO 2
    Multi Only the multi-engine

Motion Sequencing - Param Lock

  1. Hold down one or more of the 1â€“16 buttons

  2. Move a knobs or manipulate a switches. This will set the value for the steps you held down

  3. Smoothing can be configured in Sequence Edit Mode, button 12

Motion Sequencing - Real Time

  1. Press Play

  2. Press Motion Mode

  3. Press Rec

  4. Move knobs or switches. Up to four different things can be recorded

  5. Smoothing can be configured in Sequence Edit Mode, button 12

Noise Engine

  1. Set the multi-engine switch to the bottom setting

  2. Adjust the Type knob to choose the type of noise:

    Type Behavior
    High Noise with a high-pass filter via Shape
    Low Noise with a low-pass filter via Shape
    Peak Noise with a bandpass filter via Shape
    Decim Sample-rate reduced noise with Shape adjusting rate and Shift Shape adjusting keyboard tracking

Oscillator Sync

  1. Set Sync to on

  2. Make sure VCO 2 is set to a higher pitch/octave than VCO 1

Sub-types for Modulation Effects

  1. Select a modulation effect with the Select switch

  2. Hold down Shift, use Select to rotate sub-types.

Time Division

  1. Enter Sequence Edit Mode, then tap 7

  2. Step Resolution can be used to adjust how long each step is

Wet/Dry Balance for Effects

  1. Select the effect with the Mod/Rev/Del switch

  2. Hold down Shift, and turn Depth to adjust wet/dry.
