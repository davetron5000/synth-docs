Multi Output

  1. While holding down Green keep pressing A# until you've selected what you want multi to do, based on the following values of the four leds:

    Function LEDs
    Quantized Voltage
    Triangle LFO
    Random Voltage
    Decay Envelope

  2. To change the CC that can control the CV output, hold down Green and hold down A#, then send a CC value on the selected MIDI Channel (e.g. move the mod wheel of your controller).

There is no way to view the selected CC once it has been set
