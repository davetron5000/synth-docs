About

This site is not affiliated with any synth maker. I just created this instead of having hand-written notes. I find that my synths are deeper than their front panel would indicate, but some of the coolest features aren't intuitive or easy to use.

I also tend to work without a computer around, so reading PDFs on an iPhone just sucks, especially when I can't remember what words a manufacturer uses for something.

How This is Built

Each synth has a JSON file that describes the various features I tend to forget how to do. This file also documents useful MIDI CC mappings as well as some notes about odd behavior.

These JSON files are then applied to EJS and an HTML template to produce the site.
